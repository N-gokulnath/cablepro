# 📺 CablePro — Cable TV Management System

CablePro is a modern, responsive, and feature-rich management system designed specifically for local Cable TV operators. It streamlines daily operations, customer management, and payment collection with a focus on speed and ease of use.

## 📸 Preview

![Dashboard Preview](https://github.com/N-gokulnath/cablepro/blob/main/src/dashboard.png)
*High-resolution dashboard preview showing real-time analytics and management controls.*

## ✨ Key Features

- **🚀 Smart Dashboard**: Get a bird's-eye view of your business with real-time statistics including total collections, active customers, and overdue payments.
- **👥 Customer Management**: Simple and powerful customer database. Track connection dates, STB numbers, package details, and contact information.
- **💳 Payment Collection**: Streamlined "Record Payment" flow designed for field agents and desktop users. Support for multiple payment methods and billing periods.
- **📊 Financial Reports**: Generate professional collection reports to track revenue and growth over time.
- **🌓 Modern UI/UX**: Premium design with full Dark Mode support, mobile-first responsiveness, and smooth animations.
- **🔐 Secure Data**: Built-in backup reminders and local storage syncing to ensure your customer data is always safe.

## 🛠️ Built With

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS 4
- **State Management**: Zustand
- **Database**: Firebase (Authentication & Realtime Database)
- **Icons**: Material Symbols Outlined
- **Charts**: Recharts
